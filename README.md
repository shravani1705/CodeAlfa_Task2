# 📈 Stock Trading Platform

A simple Java-based Stock Trading Platform that simulates stock market operations. Users can view market data, buy and sell stocks, and track their portfolio performance.

## 🚀 Features

- View available stock market data
- Buy stocks
- Sell stocks
- Track portfolio holdings
- Monitor account balance
- Object-Oriented Programming (OOP) implementation
- Console-based user interface

## 🛠️ Technologies Used

- Java
- Eclipse IDE
- OOP Concepts

## 📂 Project Structure

StockTradingPlatform.java
├── Stock Class
├── User Class
└── Main Class

## ▶️ How to Run

1. Open Eclipse IDE
2. Create a Java Project
3. Create package:
   stocktradingplatform
4. Add `StockTradingPlatform.java`
5. Run as Java Application
